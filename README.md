## 📌 About the Project

🚀 **PayFlow Frontend** is the user interface of the PayFlow digital wallet system.

It provides a modern fintech dashboard where users can:

- Send money instantly
- Request payments
- Scan QR codes
- Track wallet activity
- Analyze transactions through charts

The interface is designed with a **clean fintech-style UI and real-time updates**.

## ✨ Key Features

### 💳 Wallet Dashboard
- Real-time wallet balance
- Transaction activity feed
- Spending analytics

### 💸 Send Money
Send payments using:

- Email
- Mobile number
- UPI ID
- QR code scanning

### 📩 Payment Requests
- Request payments
- Accept or reject requests
- Track request status

### 📊 Analytics
- Wallet spending insights
- Transaction charts
- Activity timeline

```bash
